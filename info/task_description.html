When New Year is coming everybody waits the visit of Santa Claus. He comes with a huge bag of gifts, and you can take any. But how to make the best choose - so Santa gets them out of the bag one at a time, and if you don't like it - puts it again to bag forever. What a disappointment, when the last thing from the bag is a doll, and you have already refused to pirate hat because hoping to get a saber!

When this happened with John, Dad told him - "Don't worry, I teach you how to choose a better gift. Did you knew how much gifts in the bag?"

You are given a number of gifts in a bag. The "bag" is a generator with floats -- prices for the gifts. You can iterate this generator but only once and while you iterate the "bag" you can accept a current gift by calling "accept_gift" function which is given as the third argument.

Your function will be checked many times with different bags which have various sizes from 10 to 1000 gifts. We count only the best gifts from each bag -- the second place gifts are not for us.

Input: Three arguments.
total_gifts - a number of gifts in a bag as an integer.
bag - a generator that returns a float values of gifts.
accept_gift - the callback function which you should call to accept gift.

Output: It does not matter.

Implementation examples (only to demonstrate the API): 

def choose_good_gift(total_gifts, bag, accept_gift):
    for gift in bag():
        if gift >= 1000000:  # I want million!
            accept_gift()
            return
    
def choose_good_gift(total_gifts, bag, accept_gift):
    for i, gift in enumerate(iter(bag())):
        if i == total_gifts // 2:  # on the middle
            accept_gift()
            return

How to use: For example, to choose time for catapult from the faulty aircraft - to land safely :)

Precondition:
10 ≤ total_gifts ≤ 1000
0 ≤ gift
Gifts are offered in the random order.
