Мешок Деда Мороза

С приближением Нового Года все с нетерпением ждут визита Деда Мороза. Он приходит с огромным мешком подарков, и один из них может достаться тебе. Но как выбрать лучший – ведь Дед Мороз достаёт их из мешка по одному, предлагает, а если откажешься – спрячет снова в мешок и больше уже не достанет. Как обидно, когда последним предметом из мешка оказывается кукла, а ты уже отказался от шляпы пирата, потому что надеялся заполучить шпагу!

Когда такое случилось с Ваней, папа сказал ему – не переживай, я научу тебя, как выбирать подарок получше. Ты ведь знал, сколько всего подарков в мешке?

Выбрать из потока чисел заранее известной длинны по возможности наибольшее.

Входные данные:

total_gifts - число подарков в мешке – int

bag - мешок Деда Мороза – функция-генератор, возвращает float (ценность подарка в глазах Вани, положительное)

accept – принять подарок - функция обратного вызова (callback function)

Выходные данные: Нет.

Примеры реализации (они только демонстрируют API):

Первый:

def choose_good_gift(total_gifts, bag, accept_gift):
    for gift in bag():
        if gift >= 1000000:  # I want million!
            accept_gift()
            return

Второй:

def choose_good_gift(total_gifts, bag, accept_gift):
    for i, gift in enumerate(iter(bag())):
        if i == total_gifts // 2:  # on the middle
            accept_gift()
            return

Как использовать:

Например при выборе места катапультирования лётчика из неисправного самолёта – чтобы безопасно приземлиться :)

Ограничения на входные данные:

10 <= total_gifts <= 1000

0 <= gift

Подарки извлекаются из мешка в случайном порядке.
